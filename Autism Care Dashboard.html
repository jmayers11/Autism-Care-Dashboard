How does this work?

☁️ Update Master File — overwrites Autism-Care-Master.json in your Drive with today's data. Use this every time you finish a session.

📄 Save New Version — creates a dated copy like Autism-Care-2026-06-08.json alongside the master. Good for milestones or before big changes.

⬇️ Load from Drive — pulls the master file down and restores it here. Use this when switching devices.

Files are saved in a folder called Autism Care Dashboard inside your Google Drive.

🗑️

Clear All Data

Permanently deletes everything in this browser — all logs, appointments, the child profile, and all uploaded files. Export a backup first.
